WikiPathways: Add a Description

Background

WikiPathways includes custom information displays for each pathway, including the title, author list, description and various tags. The pathway page makes it easy to edit these. Look for the pencil icon along the right margin to see what is directly editable. For example, every pathway should have a Description that provides context and a basic explanation of the key parts of the pathway. Like a figure legend, it can also define any special glyphs or color usage.

Challenge

Add a description to your tutorial pathway at WikiPathways, or update the description you entered when you uploaded the pathway. You will need to be logged in at WikiPathways in order to access the editing controls.


  • Go to WikiPathways and log in, if you haven't already.
  • Navigate to your tutorial pathway, for example by searching for it by WPID.
  • Scroll down to the Description section and click the pencil icon on the right to edit the description.
  • First, remove the existing description, which is left from the starter pathway ("Do not modify or delete. This pathway is part of the collection of content used by [https://wikipathways.github.io/academy/ WikiPathways Academy].").
  • Take some time to write a description of the pathway that would help someone understand the process depicted in the pathway.
  • Click the green checkmark to save the text.
  • Done! Now let's verify your work below.
