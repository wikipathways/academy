WikiPathways: Add a Description

Background

WikiPathways includes custom information displays for each pathway, including the title, author list, description and various tags. The pathway page makes it easy to edit these. Look for the pencil icon along the right margin to see what is directly editable. For example, every pathway should have a Description that provides context and a basic explanation of the key parts of the pathway. Like a figure legend, it can also define any special glyphs or color usage.

Challenge

Add a description to your tutorial pathway at WikiPathways. You will need to be logged in at WikiPathways in order to access the editing controls.


  • To complete this challenge, go to your tutorial pathway at WikiPathways
  • Log in, if you haven't already (see link in upper-left corner), and return to the page
  • Scroll down to the Description section
  • Take some time to write a description of the pathway that would help someone understand what is going on
  • Click the green checkmark to save the text
  • Done! Now let's verify your work below
