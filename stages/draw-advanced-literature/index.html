Add a Literature Reference - Advanced

Background

WikiPathways allows you to add citations to the overall pathway and to individual nodes and interactions, and PathVisio enables you to easily add references from PubMed. References from other sources, such as ISBN and DOI, are also allowed but have to be added manually.

Challenge

Add a DOI literature reference to your pathway.

  • Download the starter pathway here: advanced-literature-reference.gpml.
  • Open the gpml in PathVisio
  • Launch PathVisio and open the dowloaded file via File > Open.
  • Double-click on the textbox in the upper-left that includes title and organism (Pathway Information area).
  • Select the Literature tab and click the New reference button.
  • Instead of using the Query PubMed feature, we are going to just enter the DOI number in the Source field. This reference will serve as a placeholder that we can then edit manually.
  • Click Ok to exit out of the Literature reference properties interface, and save the gpml.
  • Next, open the gpml in a text editor.
  • We are going to edit the reference we just added to make it complete. For a textbook: Metabolism and Molecular Physiology of Saccharomyces Cerevisiae. At a minimum, the reference has to include the identifier (DOI 10.1201/9780203503867) and the database (in this case DOI), but we can also enter additional information. In this case, we also have the following information:
    • Title
    • Authors
    • Publication year
