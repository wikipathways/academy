WikiPathways: Add Pathway Descriptions

Goal

The goal of this task is to improve pathway descriptions to ensure high-quality descriptions on all pathways.

Instructions

  • Click the pathway image or title below to view the pathway at WikiPathways. You will need to be logged in at WikiPathways.
  • On the pathway page, click the pencil icon in the Description section, and add test to describe the pathway content. Ideally, add at least 2-3 sentences.
  • Click Next pathway to load the next pathway that needs a description.
