WikiPathways: Add Ontology Term

Background

WikiPathways uses ontologies to help categorize pathways to make them easier to find and sort. An ontology is simply a collection of terms arranged hierarchically, like a branching tree or folders with subfolders. We get our ontologies from another curated database dedicated to collecting and distributing biology-related ontologies, called BioPortal. For example, BopPortal includes a Pathway Ontology. Whenever you create or edit a pathway you should take some time to annotate it with descriptive ontology terms.

Challenge

Add an ontology term to your tutorial pathway at WikiPathways. You need to be logged in at WikiPathways in order to access the Ontology Tags controls.


  • Go to WikiPathways and log in, if you haven't already.
  • Navigate to your tutorial pathway, for example by searching for it by WPID.
  • Scroll down to the Ontology Tags section.
  • Search or browse for an ontology term you'd like to add, for example "classic metabolic pathway".
  • (Optional) Click i for more information about this ontology term.
  • Click the green checkmark to add the term to the pathways.
