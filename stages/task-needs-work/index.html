WikiPathways: Improve Pathways in Need

Background

Content at WikiPathways can always be improved and in addition to the set of task-specific curation tags there is also a "Needs work" curation tag which can be used to leave a comment about needed improvements to a pathway. The "Needs work" tag can suggest a variety of improvements not easily indicated by other curation tags, such as suggestions on layout and style, identifier data sources used etc. The goal of this task is to improve pathways that are tagged with the "Needs work" tag with the ultimate goal of including them in the Curated collection.

Instructions

  • For the pathway below, click the image or title to view/edit the pathway at WikiPathways. You will need to be logged in at WikiPathways in order to access the editing controls.
  • Using the suggestions in the "Needs work" tag as a guide, make suggested edits to the pathway.
  • Click "Verify" when you have saved the pathway.
  • Click "Next" to load the next pathway.
