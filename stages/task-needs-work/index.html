WikiPathways QA - Improve Pathways in Need

Goal

The goal of this task is to improve pathways that are in need of improvement (tagged with the Needs work tag) with the ultimate goal of including them in the Approved pathway collection. Choose from a list of prioritized tasks, or review the full list of pathways in need of improvement:

  • Prioritized curation tasks: If you select a task, please assign yourself in GitHub under Assignees.
  • Full list of pathways in need of improvement: If you select a pathway, please add it to our list of prioritized tasks in GitHub using the Improve Pathway Curation Task template, and assign yourself under Assignees.

    • Instructions

    • You will need to have PathVisio and the WikiPathways plugin installed and running.
    • To open the pathway in PathVisio, search WikiPathways for the pathway title or WPID using the WikiPathways App, under Plugins > WikiPathways > Search.
    • Using the suggestions in the GitHub issue or Needs work tag at WikiPathways as a guide, make edits to the pathway in PathVisio. To submit your changes to WikiPathways, click Plugins > WikiPathways > Update and enter a description of your edits.
    • If you've addressed all the issues, add the Approved pathway tag and remove the Needs work tag.
    • Otherwise, if issues remain, edit the Needs work tag and update the suggested edits.
