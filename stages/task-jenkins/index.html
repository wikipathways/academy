WikiPathways: Jenkins report

Goal

To work on issues from the automated Jenkins tests. These tests check for many aspects of WikiPathways curation, and the list of failed tests corresponds to potential curation tasks. The list of individual issues can be quite long; the point of this task is NOT to fix all of them; fixing even one issue is considered a success!

Unknown DataNode Type

The goal of this task is to assign Unknown data nodes to their proper type, or to convert them to graphical objects/labels.

  • On the report page, expand the entries ending in DataNodes.unknownTypes.
  • The report will include some information on the data node in question:
    http://www.wikipathways.org/instance/WP2209_r117419 http://rdf.wikipathways.org/Pathway/WP2209_r117419/DataNode/da315 (Catalyst reactions)
    In this case, the data node with label Catalyst reactions is of type Unkown.
  • Open the relevant pathway in PathVisio using the WikiPathways plugin and the WPID listed in the report, then find the problematic node, using its label.
  • If it's clear what type the node should be, assign it. If it's not a data node, redraw as Label (graphical element). Pro-tip: One can use the TypeConverter plugin to quickly change the type of a Data Node. Examples:
    • A DataNode that has been annotated with a database identifier, or that can be annotated with an identifier, should be either Gene Product, Protein, Metabolite, RNA or Pathway. For example, H2O should be a metabolite.
    • A DataNode representing a process that does not have an identifier in a public database should be converted into a Label, for example Lifespan.

Weird Characters in Titles

The goal of this task is to ensure pathway titles do not include character values that are not allowed.

  • On the report page, expand the entries ending in weirdCharacterTitles.
  • The report will include a summary for each affected title:
    http://www.wikipathways.org/instance/WP2813_r102405 'Mammary gland development pathway - Embryonic development (Stage 1 of 4)' has 2 weird characters
    In this case, the title includes two rounded brackets, which are not allowed.
  • Find the WPID for an affected pathway.
  • Go to WikiPathways and update the title to avoid special characters. See our author guidelines for details on pathway titles.

Duplicate Titles

  • On the report page, expand the entries ending in duplicateTitles.
  • The report will include a statement describing two pathways with the same title:
    http://www.wikipathways.org/instance/WP2231_r118114 'Programmed cell death and cell engulfment - Caenorhabditis elegans' is duplicate title of http://www.wikipathways.org/instance/WP2829_r118448
  • Review the two pathways to compare them. Process as follows:
    • If one pathway is just a copy of the other with no other new information, tag the latest version of the pathway for deletion.
    • If the pathways are unique, update one of the titles to better reflect the content, for example .
