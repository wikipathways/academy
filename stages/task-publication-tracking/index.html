WikiPathways: Track publications

Goal

The purpose of this task is to track relevant publications that mention specific WikiPathways models.

Instructions

  • Start by opening a tab with the latest PMC image search results for WikiPathways. This listing shows any PMC-indexed publication mentioning "WikiPathways" in the figure caption.
  • To find papers referencing specific pathway models, do a page search (Ctrl+F) for "WP". Record the PMCID ID for the publication, by clicking on the figure result to open the article. As an example, we will use PMC9603647, which references WPID4629.
  • Find the corresponding pathway on WikiPathways, for example WP4629.
  • In the Cited In section, click the pencil icon to edit the list. This will open a new browser tab with the citedin_lookup.yaml file open in edit mode.
  • Click anywhere in the editable area and use Ctrl+F to search for the WPID, for example "WP4629".
  • If the PMCID has not already been added, add an entry under the relevant WPID in the following format: - link: PMC9603647
  • Enter a commit message ("added citation for WP4629") and click Commit changes.
  • Repeat for any other WPID mentioned in the search results list.
