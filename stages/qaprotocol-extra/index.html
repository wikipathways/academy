WikiPathways: QA additional tasks

Congratulations

You are almost done! There are just a few more steps in the quality assurance protocol.

Test search

The goal of this task is to make sure that the search feature is working and the index is up-to-date.

  • Open the list of New Pathways and attempt to search for the latest pathway by entering its title into the left sidebar search tool.
  • There may be many results, but be sure that the new pathway is among them. If the search does NOT return the pathway you searched for, this means the search is broken. Please include this information in your curation report (see below).

Pathways tagged for deletion

The goal of this task is to delete any pathways that are tagged for deletion and have no recent updates.

  • If you are a sysops administrator, review the Pathways tagged for deletion.

Pass the torch

  • Submit your curation report to WikiPathways Help Discussions. If you don't have access, send your GitHub credentials to Alex or Kristina.

That's it! Thanks for helping to make WikiPathways a high-quality resource for pathway information! If you have additional time, feel free to take on additional tasks by clicking the Next step button.
